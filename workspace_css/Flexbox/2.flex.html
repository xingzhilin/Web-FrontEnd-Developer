float实现布局

float布局时，居中的div必须要放在最后。居中的div的模型宽度为整个页面，但是展示的时候会在可视范围内展示。即不会展示大屏居左或者居右的div下面。 因为float的初衷是实现文字环绕图片 ，当我们缩小浏览器的宽度时，会发现文字会环绕在居左和居右的div周围。


position实现布局

position布局时，居中的div必须放到最后。 与float不同的事，文字会被左右的div遮盖 ，所以需要使用margin-left:100px;来调整div

table + table-cell实现布局

使用display:table实现左中右时， 居中的div必须放中间 。另外，父元素必须设置width:100%; 。并且，居中的div的宽度不再为页面的宽度

flex实现布局

使用flex实现左中右时，居中的div设置flex:1，居中的div的宽度不再为页面的宽度

grid实现布局

使用grid实现左中右时，网格内部的div的宽度和高度都由父标签设置！
