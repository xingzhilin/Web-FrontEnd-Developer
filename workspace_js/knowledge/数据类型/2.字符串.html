String - 字符串

多行字符串

`qweq qweqwe qweqwe`

模板字符串

字符串拼接

var name = '小明'; var age = 20; var message = '你好, ' + name + ', 你今年' + age + '岁了!'; alert(message);

es6 模板字符串 ${}

var name = '小明'; var age = 20; var message = `你好, ${name}, 你今年${age} 岁了!`; alert(message);

console.log(s.toLowerCase())

toLowerCase

console.log(s.toLowerCase())

indexOf

indexOf()会搜索指定字符串出现的位置： console.log(s.indexOf('world'))
