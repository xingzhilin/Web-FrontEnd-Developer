# html-webpack-plugin
    插件： 
    安装： cnpm install html-webpack-plugin –save-dev
    编辑配置：
        打开： webpack.config.js
            引入 const htmlWebpackPlugin = require('html-webpack-plugin');
            增加：
            plugins： [
                new htmlWebpackPlugin()
            ]
    最后我们执行webpack的时候  会自动生成 index.html
